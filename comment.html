監修者からのひとこと

今村 久司（脳神経内科 専門医）

最新AIを用いて医学情報を整理し発信する今村久司の自己満足のページにわざわざご足労いただき、感謝いたします。今後も、色々なことに挑戦していきたいです。

ブログを始めたのは2025年3月でした。きっかけは、ChatGPTのDeep Researchの結果に感動したこと。AIのすごさを世の中に知らしめる伝道師になりたい、という思いからのスタートでした。

以降は、日常診療であたった疑問を中心に、そしてときに、診療と関係のない世の中の話題や、日常の疑問なども扱いながら、「知りたいことをAIで情報整理して発信する」という流れで、可能な範囲で継続してきました。

そんな中、AIの進化でスライド生成や音声の生成、動画化も進化するにつけ、発信の場はブログと並行してYouTubeにも広がりました。2025年の秋頃から少しずつ動画を投稿していたのですが、ついにチャンネル登録1000人を達成しました。半年間の目標だったので、ご報告させてください。

最近では、Claude Code の Fable 5、ChatGPT 5.6 sol、Gemini を合わせ技で用いて、ブログとYouTubeを運営しています。次の目標はまた模索しますが、発信活動はより自由にやっていきたいと思っています。新しいAIができて、できることが増えたなら、積極的に導入して、Qualityアップを図っていきます。
